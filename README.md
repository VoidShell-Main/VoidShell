**>DESCRIPTION / INFO**

*VoidShell is a portable, electronic tool used for a multitude of different tasks, including, but not limited to: controlling Bluetooth devices - with permissions -, accessing networks like WiFi and internet - may require a password -, running a custom made coding language (more details under section titled, "Coding & Terminal"), and many more.*
